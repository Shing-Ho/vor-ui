# vor-ui

## Install project

Install the Node packages and dependencies
```
npm run install
````
## Running project
Migrate the database
```
npm run db:migrate
```

Run the service
```
npm run service
```

Run the frontend UI
```
npm run start
```

Deploy the XYDistribute contract
```
truffle migrate --network=develop
```

Test and increase allowance of contract
```
npm run test:distribute
```
